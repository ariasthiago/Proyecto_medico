# 🏥 Sistema Web EPS - Proyecto Médico

Aplicación web desarrollada como proyecto académico durante mi formación en Análisis y Desarrollo de Software. Este sistema simula el funcionamiento básico de una EPS, permitiendo la gestión de usuarios y roles dentro de la plataforma.

---

## 🚀 Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- React
- MySQL
- Power BI (conocimiento básico aplicado para análisis de datos)

---

## 🎯 Funcionalidades principales

- Gestión de usuarios (crear, listar, editar y eliminar)
- Sistema de roles (administrador, médico, usuario)
- Autenticación básica simulada
- Interfaz amigable e interactiva
- Organización de información tipo sistema de salud

---

## 💻 Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

```bash
# Clonar el repositorio
git clone https://github.com/ariastiago/Proyecto_medico.git

# Entrar a la carpeta del proyecto
cd Proyecto_medico

# Instalar dependencias
npm install

# Ejecutar el proyecto
npm run dev
